# Microsoft.Extensions.Logging samples

This directory contains samples to show how to integrate elmah.io into Microsoft.Extensions.Logging.

## Samples list

* **Elmah.Io.Extensions.Logging.AspNetCore.Net10** - Integrate elmah.io with Microsoft.Extensions.Logging in ASP.NET Core 10.0.
* **Elmah.Io.Extensions.Logging.AspNetCore80** - Integrate elmah.io with Microsoft.Extensions.Logging in ASP.NET Core 8.0.
* **Elmah.Io.Extensions.Logging.AspNetCore80.SignalR** - Integrate elmah.io with Microsoft.Extensions.Logging in ASP.NET Core 8.0 running SignalR.
* **Elmah.Io.Extensions.Logging.AspNetCore90** - Integrate elmah.io with Microsoft.Extensions.Logging in ASP.NET Core 9.0.
* **Elmah.Io.Extensions.Logging.Console.Net10** - Integrate elmah.io with Microsoft.Extensions.Logging in a .NET 10.0 console application.
* **Elmah.Io.Extensions.Logging.Console80** - Integrate elmah.io with Microsoft.Extensions.Logging in a .NET 8.0 console application.
* **Elmah.Io.Extensions.Logging.Console90** - Integrate elmah.io with Microsoft.Extensions.Logging in a .NET 9.0 console application.
* **Elmah.Io.Extensions.Logging.EntityFrameworkCore.Net10** - Integrate elmah.io with Microsoft.Extensions.Logging in a .NET 10.0 console application with EntityFramework Core.
* **Elmah.Io.Extensions.Logging.Function** - Integrate elmah.io with Azure Functions in .NET 8.0.
* **Elmah.Io.Extensions.Logging.Function.Isolated** - Integrate elmah.io with Isolated Azure Functions in .NET 10.0.
